var: Can be redeclared and reassigned.
let: Cannot be redeclared but can be reassigned.
const: Cannot be redeclared or reassigned.
Spread Operator (...): Expands elements of an array or properties of an object.
map(): Transforms each element and returns a new array.
filter(): Returns elements that satisfy a condition.
forEach(): Executes a function for each element without returning a new array.
Arrow Function: A shorter syntax for writing functions using =>.
Template Literals: Strings written with backticks that allow variable interpolation using ${}.
